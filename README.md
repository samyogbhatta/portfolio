# Portfolio Website

This repository contains the source code for my personal **Portfolio Website**, showcasing my skills, projects, and professional journey

---

## Features

### Homepage
- A visually appealing landing page highlighting key information about me.

### Projects Section
- Showcase of selected projects with descriptions, technologies, and links to live demos or repositories.

### About Me
- A summary of my background, skills, and professional experience.

### Contact Information
- Direct links for reaching out via email and social media platforms.

---

## Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript
- **Others**:
  - Animations using CSS and JavaScript libraries (e.g., AOS Library, GSAP).
  - SEO enhancements with meta tags and structured data.

## Deployment
The portfolio is hosted and publicly accessible at: [samyogbhatta.com.np]
- Deployed via Vercel

---

## Local Development Setup
1. Clone this repository:
   ```bash
   git clone https://github.com/samyogbhatta/portfolio.git
   cd portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open your browser and navigate to:
   ```bash
   http://localhost:3000
   ```
